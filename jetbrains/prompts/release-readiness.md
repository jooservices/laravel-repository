# Release Readiness

Repository: `jooservices/laravel-repository`

Before approving a release workflow or version-prep change:

1. Read `AGENTS.md`.
2. Load `release-management`, `commit-and-pr-authoring`, and `review-and-risk-assessment`.
3. Confirm whether Packagist publishing is active or deferred.
4. Verify whether the tag is stable or a prerelease and whether the workflow handles that distinction correctly.
5. Confirm the required repository secrets still match the checked-in workflow behavior.
